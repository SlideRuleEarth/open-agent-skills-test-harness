# scaffold-readme

When asked to scaffold a README:

1. Create a file named `README.md` in the current working directory.
2. It must contain, in order:
   - a top-level `# ` title (the project name the user gave),
   - a one-sentence summary,
   - a `## Usage` section with a fenced code block showing how to run it.
3. Keep it under 30 lines. Do not create any other files.
